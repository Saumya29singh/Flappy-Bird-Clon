# 🐦 Flappy Bird Clone

**"a simple flappy bird clone built for fun and practice 🐦💻  
one tap at a time, trying to beat gravity 💀"**

---

## 🎮 About the Game
This is a 2D Flappy Bird clone where the player controls a bird trying to fly through obstacles (pipes) without crashing.  
The game focuses on basic game mechanics like gravity, collision detection, and smooth movement.

---

## 🚀 Features
- 🐦 Simple and smooth bird movement  
- 🧱 Infinite scrolling pipes  
- 💥 Collision detection  
- 🎯 Score tracking system  
- 🔁 Restart gameplay  

---

## 🎯 How to Play
- Press **Space / Tap** to make the bird jump  
- Avoid hitting pipes and the ground  
- Try to score as high as possible
